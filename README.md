# **新云天明的童话**

---

## 一、灯

城里有一盏灯。

不是普通的灯。这盏灯很亮，亮到可以把人的影子照得很长很长。城里的人都说，只要站在灯下，就不会冷。

林晚第一次听说这盏灯，是小时候。奶奶说，等你长大了，就能看见那盏灯。她问，灯在哪里？奶奶说，在城中心。每天晚上都亮。

后来她真的进城了。城很大，人很多，她找了很久，终于找到那盏灯。

灯确实亮。灯下也确实暖。很多人站在灯下，互相靠着，互相抱着，互相说“你来了”。她也站进去，被人抱了一下。抱她的人她不认识，但那人的手很暖。她在那双手里，忽然想哭。

那天晚上她没走。灯一直亮着，她就一直站着。身边的人换了一拨又一拨，但灯没灭，暖没散。

后来她知道，站在灯下是要钱的。不是直接要，是别的办法。比如，灯会发一种小牌子，牌子上的图案会变。有的人收集很多牌子，挂满全身，灯就会照得他们更亮一些。也有人戴一种很大的帽子，帽子会发光，戴上之后，灯就不那么刺眼了。

她开始攒钱。

攒钱的办法有很多种：少吃一顿饭，少坐一趟车，少买一件衣服，少说一句话。她把每天省下来的钱存在一个盒子里，盒子上画了一盏灯。

攒了很久。久到她忘了具体多久。有一天盒子满了，她打开数了数，够买一个发光的小牌子了。

那天晚上，她站在灯下，把牌子挂在胸前。牌子亮了，灯也亮了。身边的人看了她一眼，有人说：牌子好看。她点点头，没说话。

但那天晚上，她忽然想：这个牌子，是换什么的？

换来那一眼？换来那句话？换来灯下多站一会儿？

她不知道。但她继续站着。灯一直亮，暖一直有。

后来她攒够了钱，买了第二块牌子。又攒，买第三块。攒，买。攒，买。

有一天，她低头看胸前的牌子，发现已经挂了七个。七个牌子叠在一起，沉甸甸的，压得她有点喘不过气。她抬头看灯，灯还是那么亮。她看身边的人，人还是那么多。

她忽然想问一个人：你攒了多久？

但她没问。因为那个人也在低头看自己的牌子。

那天晚上回去，她打开那个画着灯的盒子。盒子空了。她又开始往里放钱。

放的时候她想起奶奶。想起奶奶说，等你长大了，就能看见那盏灯。

她看见了吗？看见了。

然后呢？

没有然后。

---

## 二、河

城外有一条河。

河很宽，宽到看不见对岸。河水流得很慢，慢到像没有在动。但每个站在河边的人都知道，它一直在流。

林晚第一次去河边，是跟一个人去的。那个人她在灯下认识的，叫阿远。阿远说，城里太吵了，去河边坐坐吧。

她去了。河边很静。只有水流的声音，很轻，轻到要屏住呼吸才能听见。

阿远坐在河边，看着对岸。对岸什么也看不见，只有雾。她问：你在看什么？阿远说：不知道。但她知道阿远在看什么。

那天晚上，他们在河边坐了很久。阿远说了很多话。说小时候，说老家，说来城里之后的事。说灯很亮，但站久了腿会酸。说牌子很漂亮，但多了会重。说人很多，但有时候还是会觉得——后面的话没说完，停了。

林晚没问。她只是听着。

后来天黑了，他们回城。灯还亮着。阿远又站进灯下，又有人抱他，他又开始笑。林晚站在旁边看了一会儿，然后回家了。

那天晚上，她做了一个梦。梦里她站在河边，对岸没有雾，看得很清楚。对岸站着很多人，都是她不认识的。那些人也在看她。她想喊，喊不出声。她想过去，过不去。河太宽了。

醒来之后，她想起阿远。想起阿远看对岸的样子。想起那句没说完的话。

后来她又去过几次河边。有时候一个人，有时候和阿远一起。每次去，阿远都看着对岸。每次看，对岸都是雾。

有一次她问：你觉得对岸有人吗？

阿远说：有吧。

她说：你怎么知道？

阿远说：不知道。但如果有的话，他们也在看这边吧。

她没说话。看着河。河水一直在流。

那天回去之后，她又站在灯下。灯很亮，人很多，有人抱她。但她忽然觉得，抱她的那双手，有点远。

不是距离的远。是别的远。

她说不清。

---

## 三、城

城里有很多人。每天都有新的人来，每天都有旧的人走。

林晚在城里住了很久。久到她已经数不清自己买了多少块牌子，在灯下站了多少个晚上。

她见过很多事。

见过一个人站在灯下哭，哭完了又笑，笑完了又哭。旁边的人拍着她的背，说没事，在这里可以哭。她就一直哭。

见过两个人吵架，吵得很大声，所有人都看着。吵完之后，两个人又抱在一起，说对不起。然后继续站在灯下。

见过一个新来的人，第一次站在灯下，被人抱了一下，整个人都僵了。然后她看见那人眼睛红了，但没哭。

见过一个老来的人，胸前挂满牌子，走起路来哗哗响。所有人都认识他，所有人都喊他的名字。有一天他不来了。后来有人说起他，说回老家了。再后来就没人提了。

她也见过灯。

灯一直亮着。不管白天黑夜，不管人来人往。灯就那样亮着。有时候她觉得灯很近，近到伸手就能摸到。有时候她觉得灯很远，远到像在天上。

有一天晚上，她站在灯下，忽然想起很久以前的事。想起奶奶说的那句话。想起第一次来城里的那天。想起攒钱买第一块牌子的那些日子。想起阿远第一次带她去河边。想起那个在灯下哭的人。想起那个回老家的人。

她想起这些事，然后发现自己站了很久。久到身边的人换了好几拨，久到灯还是那么亮。

她低头看胸前的牌子。七个。沉沉的。

她忽然想：这七个牌子，换来了什么？

换来了灯下的暖。换来了有人抱她。换来了有人喊她的名字。换来了那几千个站在灯下的晚上。

但她又想起河边。想起阿远看对岸的样子。想起河一直在流。

她忽然想问一个问题：如果有一天灯灭了，她会去哪？

没人能回答。

那天晚上，她没在灯下站太久。她回了家，打开那个画着灯的盒子。盒子又满了。她数了数，够买第八块牌子。

但她没买。

她把盒子盖上，放在一边。然后躺下。窗外的夜很黑。城里很静。灯还在亮着，隔着很远，她看不见，但知道它在亮。

她想起河。想起对岸。想起阿远说的那句：如果有的话，他们也在看这边吧。

她忽然想：也许对岸真的有人。也许那些人也在看这边。也许他们看的是灯，也许看的是别的。

也许他们看的，是这条河。

河一直在流。从他们那边，流到这边。从这边，流到那边。

灯会灭，人会走，牌子会旧。但河一直在流。

---

## 四、后来

很久以后，林晚离开那座城。

走之前，她又去了一次河边。那天雾很大，对岸什么也看不见。她在河边坐了很久，久到天快黑了。

走的时候，她把那个画着灯的盒子留在了河边。盒子里有她攒了很久的钱，够买第八块牌子。

她不知道谁会捡到。捡到的人会不会拿去换牌子。会不会站在灯下。会不会也有一天来河边坐坐。

不知道。但她想，如果有人来，也许会看见这条河。

河一直在流。

进城的人看不见它。站在灯下的人看不见它。挂满牌子的人看不见它。

但河一直在流。

她走的时候，天黑了。城里的灯又亮了，隔着很远，能看见一点光。但她没回头。

她沿着河边走。不知道去哪。但河水一直流，她就一直走。

走着走着，她忽然想起很久以前，奶奶说的那句话：等你长大了，就能看见那盏灯。

她看见了吗？看见了。

然后呢？

然后她看见了河。

---

## 五、尾声

这个故事没有结尾。

城里灯还亮着。每天都有人来，每天都有人走。有人攒钱买牌子，有人站在灯下哭，有人互相抱着说“你来了”。

城外河还流着。每天都有人来河边坐坐，每天都有人看着对岸。对岸永远是雾，河永远在流。

林晚去了哪，没人知道。

但有人说过，如果你沿着河边一直走，一直走，走到很远很远的地方，也许会看见一个人。那个人坐在河边，看着对岸。她胸前没有牌子。她身后没有灯。她只是坐着，看着。

如果你问她看什么，她不会回答。

但如果你也坐下来，看一会儿，你可能会发现：对岸有没有人，其实不重要。

重要的是河一直在流。

而坐在河边的人，终于可以喘一口气了。
